# GameLibrary
React + dotnet core project to have a game library

## Live Demo

The React frontend is deployed on GitHub Pages: [https://farhan-rasheed-chughtai.github.io/GameLibrary/](https://farhan-rasheed-chughtai.github.io/GameLibrary/)

## Project Structure

- `FrontEndReact/` - React TypeScript frontend application
- `BackEndDotNetCore/` - .NET Core backend API

## Deployment

The frontend is automatically deployed to GitHub Pages on every push to the `main` branch via GitHub Actions.
